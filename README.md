# 📱 Motorola Sales Dashboard | Power BI

## 📌 Project Overview

This project is an interactive **Motorola Sales Dashboard** developed in **Power BI** to analyze sales performance across different cities, mobile models, customer ratings, payment methods, and monthly trends.

The dashboard enables users to monitor key business metrics and gain insights through interactive visualizations and filters.

---

## 🎯 Objectives

- Analyze overall sales performance
- Track sales across different cities
- Compare sales by mobile model
- Monitor customer ratings
- Analyze payment method preferences
- Observe monthly sales trends
- Provide an interactive business dashboard for decision-making

---

## 🛠️ Tools & Technologies

- Power BI
- Power Query
- DAX
- Microsoft Excel / CSV Dataset

---

## 📊 Dashboard Features

### KPI Cards
- Total Sales
- Total Quantity Sold
- Total Transactions
- Average Sales Value

### Interactive Slicer
- Month-wise Filter (January–December)

### Visualizations
- 🗺️ Total Sales by City (Map)
- 📈 Total Quantity by Month
- ⭐ Customer Ratings Distribution
- 💳 Transactions by Payment Method
- 📅 Total Sales by Day Name
- 📱 Total Sales by Mobile Model
- 📋 Brand-wise Sales Summary Table

---

## 📈 Key Insights

- Samsung generated over **6M** in total sales.
- The dashboard tracks sales across multiple Indian cities.
- UPI, Credit Card, Cash, and Debit Card payment methods are compared.
- Mobile models can be analyzed based on their sales performance.
- Monthly filtering allows quick comparison of business performance.
